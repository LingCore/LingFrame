# LingFrame

LingFrame —— Windows 屏幕录制与编辑工具 / Windows screen recorder & editor.

## 下载 / Download

最新安装包请到 [Releases](https://github.com/LingCore/LingFrame/releases) 页面下载。

Get the latest installer from the [Releases](https://github.com/LingCore/LingFrame/releases) page.

- **系统 / System**: Windows 10 / 11 (x64)
- **最新版 / Latest**: [LingFrame v0.1.1](https://github.com/LingCore/LingFrame/releases/latest)

## 主要功能 / Features

- 高性能屏幕捕获（DDA / WGC，支持 HDR）/ High-performance screen capture (DDA / WGC, HDR-aware)
- GPU 加速视频导出（D3D11 + NVENC / AMF / QSV）/ GPU-accelerated export
- 光标平滑追踪、智能缩放、绘图叠加 / Cursor smoothing, smart zoom, drawing overlay
- 桌面 + 麦克风混音 / Desktop + microphone audio mixing
- Whisper 语音字幕（CUDA / Vulkan / CPU 自适应）/ Whisper-based subtitles (CUDA / Vulkan / CPU)
- Material 3 Expressive 风格 UI / Material 3 Expressive UI
